Nếu bạn chỉ cần tắt thanh cuộn (overflow x và y) của <textarea>, dùng thẳng Tailwind hoặc CSS như sau:

Tailwind CSS

html

Sao chép
Chỉnh sửa 
                                
<textarea
        class="w-full overflow-hidden resize-none border-none outline-none focus:ring-0 p-2 bg-gray-100 focus:ring-0 p-2 bg-gray-100 focus:ring-0 p-2 bg-gray-100 focus:ring-0 p-2 bg-gray-100 focus:ring-0 p-2 bg-gray-100"
        rows="2"
        placeholder="Hỏi bất cứ điều gì"></textarea>
  • overflow-hidden: tắt thanh cuộn ngang và dọc.
  • resize-none: không cho người dùng kéo giãn.

CSS Thuần

html

Sao chép
Chỉnh sửa 
                                                
                textarea {
                  overflow-x: hidden;  /* tắt cuộn ngang */
                  overflow-y: hidden;  /* tắt cuộn dọc */
                  resize: none;        /* tắt resize */
                  border: none;
                  outline: none;
                }

Lưu ý:

  • Khi tắt overflow, nếu người dùng gõ quá dài, phần text sẽ bị ẩn (không cuộn, không resize).
  • Nếu bạn muốn giống ChatGPT (tự động giãn chiều cao khi gõ, không cuộn), cần thêm JS để auto-resize.
